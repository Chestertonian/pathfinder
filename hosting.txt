# 🧠 High-level idea (important)

Right now:

```text
Player A terminal → local DB → game loop
Player B terminal → local DB → game loop
```

You want:

```text
Player A PC → shared DB (your machine)
Player B PC → shared DB (your machine)
Player C PC → shared DB (your machine)
```

So your computer becomes:

> 🧠 the world state server (PostgreSQL host)

But NOT the game server logic.

# 🧱 STEP-BY-STEP SETUP

## 🟢 STEP 1 — Make PostgreSQL accessible on your network

On your machine (the host):

### 1. Find PostgreSQL config files

Usually:

* `postgresql.conf`
* `pg_hba.conf`

---

## 🟡 STEP 2 — Allow external connections

### In `postgresql.conf`

Find:

```conf
listen_addresses = 'localhost'
```

Change to:

```conf
listen_addresses = '*'
```

This allows other machines to connect.

---

## 🟡 STEP 3 — Allow your LAN subnet

In `pg_hba.conf`, add:

```conf
host    all     all     192.168.1.0/24     md5
```

(Adjust subnet if your network differs)

---

## 🟡 STEP 4 — Restart PostgreSQL

Windows:

* Services → PostgreSQL → Restart

Linux:

```bash
sudo systemctl restart postgresql
```

---

## 🟢 STEP 5 — Find your local IP

On your host machine:

Windows:

```bash
ipconfig
```

Look for:

```text
IPv4 Address: 192.168.1.42
```

That is your **server address**

---

## 🟢 STEP 6 — Open firewall port

Allow:

```text
5432 (PostgreSQL default)
```

Windows will prompt you — allow Private Networks.

---

## 🟢 STEP 7 — Update your game connection string

In `db.py` (or wherever you connect):

### BEFORE:

```python
host="localhost"
```

### AFTER:

```python
host="192.168.1.42"
```

(Your machine’s IP)

---

## 🟢 STEP 8 — Create shared DB user (IMPORTANT)

You need a DB user other machines can log in with:

```sql
CREATE USER game_user WITH PASSWORD 'securepassword';
GRANT ALL PRIVILEGES ON DATABASE your_db TO game_user;
```

---

## 🟢 STEP 9 — Distribute game to other players

Each player:

* clones/downloads your Python project
* installs dependencies (`pip install`)
* runs:

```bash
python main.py
```

BUT with DB config pointing to YOUR IP.
